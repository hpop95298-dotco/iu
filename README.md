# Innovation University Platform
## A comprehensive digital ecosystem for modern education

Built with: **HTML5 | CSS3 (Glassmorphism) | Vanilla JavaScript | Chart.js**

### 🌟 Features
- Premium Navy & Gold Design System
- Student Dashboard with performance analytics
- Multi-level course management
- Attendance & Schedule management
- Responsive for all devices

### 🚀 How to Run Locally
```bash
npx serve Home1
```
Then navigate to `http://localhost:PORT/pages/iu.html`
